Portfolio Project

I developed a personal portfolio website that is designed to present my career history, technical skills, achievements, academic credentials, and project experiences in a clear and well-structured format. The website is not just a tool for self-branding but also a professional website where recruiters and potential clients can just scan my proficiency, determine the quality of work, and realize my accomplishments. In addition, it provides an accessible contact channel, which makes communication and cooperation easier and more efficient.

Role

  • Full-stack / front-end developer

Technologies

  • Frontend: HTML, CSS
  • Hosting : Github Pages

Key Features

  • Homepage introducing personal profile and vision
  • Skills section highlighting both technical and creative tools
  • Achievements section presenting academic rankings and awards
  • Experience section including internships as a Graphic Designer and Computer Technician
  • Education section showcasing academic background
  • Project showcase with visuals and descriptions
  • Simple, responsive, and user-friendly navigation

Results

  • Live website accessible at: https://nbillfyyad.github.io/Nabil-s-Portfolio-/
  • Clean and minimalistic interface optimized for readability
  • Easy access to contact information
